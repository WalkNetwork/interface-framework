abstract var scrollableEngines: ScrollableEngines
Content copied to clipboard

All possible scrollable engines. This is used to map all engines in this scrollable graphical interface.

Note:

This is different of engineStack because engineStack just stores all current engines, so for example, if i have a scrollable graphical interface with two pages and my current page is one, the engineStack only will shows the engines of my current page, while scrollableEngines will show all pages. This not includes scrollUpEngine and scrollDownEngine engine. To not includes a scrollable engine in the mapper, just adds the isPersistent metadata to a engine.
