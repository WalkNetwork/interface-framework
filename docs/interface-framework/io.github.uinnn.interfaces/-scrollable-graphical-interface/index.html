ScrollableGraphicalInterface

interface ScrollableGraphicalInterface : GraphicalInterface, Scrollable
Content copied to clipboard

Represents a scrollable graphical interface. This interface can map all scrollableEngines with schematic and mapper. This is equals to a paginated inventory. This not stores all pages in a separed graphical interface in a list, this map in runtime all scrollableEngines and the schematic and mapper will decide when showing a engine.
