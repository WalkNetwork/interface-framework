Engine

open class Engine : ItemStack, Alterable, Renderable, Pressable, Visible, Metadatable, Workable, Scrollable
Content copied to clipboard

A engine is a more extensible ItemStack. Supporting everything when working with GraphicalInterface and ScrollableGraphicalInterface. This class supports:

  • Renders Renderable

  • Press Pressable

  • Alter Alterable

  • Visibility Visible

  • Metadata Metadatable

  • Work Workable

  • Scroll Scrollable
